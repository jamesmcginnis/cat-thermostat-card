# CAT Thermostat Card

A dynamic thermostat card for Home Assistant with animated mode icons, per-mode gradient backgrounds, full colour customisation via the Visual Editor, and +/− temperature controls.

## What’s included

- Unique animated icons for every HVAC mode (heating, cooling, heat/cool, dry, fan only, off)
- Two-colour gradient background that changes automatically with the active mode
- Tap the mode icon to toggle power on/off — configure exactly which mode it powers on to
- +/− buttons to adjust target temperature in 0.5° increments
- Full Visual Editor with colour pickers, gradient previews, icon selectors, and toggle controls
- All colours, gradients, and icons are individually customisable — no YAML required
